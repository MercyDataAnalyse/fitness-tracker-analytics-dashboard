# fitness-tracker-analytics-dashboard
Data analysis and visualization project completed during DevAlpha internship
GitHub Project Report (README.md)
Fitness Tracker Analytics Dashboard | Power BI
Project Overview

This project was developed as part of my DevAlpha Internship (Task 2: Visualization Study). The objective was to transform raw fitness tracker data into meaningful visual insights using Power BI and present findings professionally.

The dashboard provides a comprehensive analysis of user fitness behavior, activity levels, calories burned, sleep patterns, heart rate trends, and exercise distribution.

Tools Used
Microsoft Power BI
Microsoft Excel
DAX (Data Analysis Expressions)
Business Problem

Fitness tracking devices generate large volumes of health and activity data. However, raw data alone does not provide actionable insights. This project aims to visualize key performance indicators and behavioral trends that support informed fitness and wellness decisions.

Key Insights
1. Cardio Dominates User Activity
Cardio exercises account for the highest proportion of recorded activities.

2. Moderate Activity Users Form the Largest Segment

Over 50% of users fall within the Moderate Activity category.

3. Higher Activity Leads to Greater Energy Expenditure

Users with higher activity levels consistently burn more calories.

4. Healthy Sleep Patterns

Average sleep duration of 7.4 hours aligns with recommended sleep guidelines.

5. Heart Rate Increases with Activity Intensity

Users in the High Activity group recorded the highest average heart rates.

Recommendations
1. Implement personalized step goals.
2. Introduce activity streaks and rewards.
3. Integrate sleep quality monitoring.
4. Provide adaptive fitness recommendations based on activity behavior.
5. Use heart-rate insights to optimize training plans.
